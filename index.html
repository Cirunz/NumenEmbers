  • Might
    Total:
    Current: Edge:
    Total value is the maximum value for this stat, when the character is fully restored.
    Current value is the current amount; +/- buttons, as long as effort, affect this value.
    Edge is the discount the character get, on the total expense of an effort action.
  • Speed
    Total:
    Current: Edge:
    +1/-1 and +5/-5 buttons make a direct change to the current value, ignoring Edge value. Effort shows a panel with additional data, to calculate stat expense to lower difficulty of an action.
    In this matter, Edge is subtracted from final expense for the effort.
    Reset button reset the current value for the stat to total value.

  • Recovery
    Bonus:
    Extras:
    Next:
    1 Action
    The base recovery roll bonus is added to a rolled D6 everytime the character perform a recovery roll.
    Extras is the amount of extra recovery action character may perform daily; after that, next roll will take 10 minutes
    Next is the time the next recovery roll will take, comprensive of residual extra actions.
Reset all set all the stats current values to their total values, and the daily recovery roll to the first of the day (1 action roll).
Does not affect shins value.
